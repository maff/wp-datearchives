A simple plugin for showing post archives categorized by date.

== Description ==

See the [blog post](http://maff.ailoo.net/2008/07/wordpress-plugin-datearchives/) for more details on usage.

== Installation ==

1. Upload `datearchives.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In your pages/posts call the plugin with `[DateArchives]`
1. In your templates call the plugin with `<?php echo DateArchives() ?>`
1. For more details (parameters) see the [blog post](http://maff.ailoo.net/2008/07/wordpress-plugin-datearchives/)
